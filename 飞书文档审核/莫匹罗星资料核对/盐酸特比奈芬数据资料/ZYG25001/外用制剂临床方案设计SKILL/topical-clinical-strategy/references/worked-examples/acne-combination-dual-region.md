# Worked Example: Acne Combination, Dual Region

## Representative Input

```text
请基于以下信息输出中国和 FDA 的临床开发策略，并比较保守路径与激进路径：三联外用痤疮凝胶，三个活性成分均已有外用先例，目标人群为 12 岁及以上，已有局部耐受、贴敷致敏和最大使用 PK 数据，拟主打提高依从性和疗效。
```

## Why This Example Matters

This example is useful for:

- fixed-dose or multi-component acne products
- dual-region strategy
- cases where max-use PK already exists
- cases where contribution logic may be unavoidable

## What a Strong Answer Should Include

- 明确指出该项目更接近改良型新药或 505(b)(2) 组合开发逻辑，而不是简单沿用单成分路径
- 明确说明成分贡献问题不能被“已有外用先例”直接替代
- 结合现有 max-use PK，说明激进路径为什么可以更积极，但不能完全跳过组合有效性论证
- 对中美路径分别说明：
  - 中国更强调临床优势主张
  - FDA 更强调组合贡献、桥接和审评先例

## Example Output Shape

### 项目结论摘要

- 当前更接近：改良型新药 / 505(b)(2) 组合型路径
- 推荐主路径：偏保守
- 最大监管风险：成分贡献证据不足时，组合便利性不足以替代组合合理性论证

### 判断级别与关键假设

- 当前判断级别：正式策略判断
- 关键假设：已有 max-use PK 可证明系统暴露未显著高于已知局部产品
- 最高优先级缺口：组合中每个活性成分的临床贡献证据

### 保守路径

- 增加明确的成分贡献研究
- 保留较完整的剂量或组合探索
- 在双区域下更稳妥

### 激进路径

- 充分利用现有 max-use PK 和已知外用成分经验
- 尽量压缩早期系统研究
- 但仍保留最核心的组合贡献论证

### 关键依据类型

- FDA 审评案例：Cabtreo
- FDA 路径 guidance
- 中国改良型新药临床试验指导原则

## Main Teaching Point

For multi-component acne products, the aggressive path can reduce PK burden more easily than it can remove contribution burden.
