# Worked Example: Minoxidil Optimization, Dual Region

## Representative Input

```text
已有外用 minoxidil 使用基础，拟开发一个改良剂型以提高依从性并降低刺激，请给出中国和 FDA 双路径开发建议。
```

## Why This Example Matters

This example is useful for:

- business-near AGA optimization programs
- strong-knowledge-base products where the aggressive path is more plausible
- cases where convenience and tolerability are the main claimed advantages

## What a Strong Answer Should Include

- 明确指出这是“优化型开发”，重点不是重新证明 minoxidil 的基础机制，而是证明新剂型确实带来依从性或耐受性收益
- 说明 AGA 的评估周期仍然较长，不能因为已有活性成分经验就过度缩短观察
- 解释为什么这类项目比 topical finasteride 类项目更适合讨论激进路径
- 对双区域路径说明：
  - 中国侧仍需要把临床优势讲清楚
  - FDA 侧可以更多利用既有外用经验，但仍要重视客观毛发评估方法

## Example Output Shape

### 项目结论摘要

- 当前更接近：改良型新药 / 505(b)(2) 优化型路径
- 推荐主路径：激进路径可讨论，但保守路径仍需保留
- 最大监管风险：如果“降低刺激/提高依从性”的主张没有对应证据，路径会被迫回到更传统的长期验证

### 判断级别与关键假设

- 当前判断级别：正式策略判断
- 关键假设：新剂型能在不削弱有效暴露的情况下改善使用体验
- 高优先级缺口：标准化毛发评估、长期依从性与耐受性支持

### 保守路径

- 更完整的剂量和观察周期设计
- 更扎实的客观毛发终点验证

### 激进路径

- 充分利用既有 topical minoxidil 经验
- 更聚焦剂型优化目标，而不是重新展开全套早期论证

### 关键依据类型

- 中国改良型新药临床优势逻辑
- AGA 模块与暴露-效应逻辑
- 已知 topical minoxidil 使用基础

## Main Teaching Point

Minoxidil-optimization programs are one of the cleaner places to allow an aggressive path, but only if the benefit claim matches the actual evidence package.
