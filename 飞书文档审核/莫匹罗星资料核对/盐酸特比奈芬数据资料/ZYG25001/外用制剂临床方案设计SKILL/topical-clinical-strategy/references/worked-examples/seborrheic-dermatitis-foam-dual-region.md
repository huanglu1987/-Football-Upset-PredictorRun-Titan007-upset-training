# Worked Example: Seborrheic Dermatitis Foam, Dual Region

## Representative Input

```text
某外用泡沫剂拟用于脂溢性皮炎，计划先美国后中国，也希望后续能支持双区域开发。目标覆盖头皮和面部，主打起效快和依从性提升。请输出保守路径和激进路径。
```

## Why This Example Matters

This example is useful for:

- one of the newly added indication modules
- scalp and face site-coverage questions
- chronic relapsing dermatitis rather than fungal-cure logic

## What a Strong Answer Should Include

- 明确先界定项目主张到底是：
  - 急性控制
  - 复发控制
  - 长期维持
- 明确头皮和面部给药的可用性、本地耐受和依从性为什么是路径核心
- 指出脂溢性皮炎项目不能简单套用浅表真菌“治愈”逻辑
- 对美国优先路径说明：
  - 近期公开审评资料对泡沫剂和慢病使用场景有较高参考价值
- 对中国路径说明：
  - 需要把临床价值和适用场景讲清楚，尤其是如果项目定位不只是“换个剂型”

## Example Output Shape

### 项目结论摘要

- 当前更接近：已知外用皮肤病路径上的差异化或优化型开发
- 推荐主路径：中度偏保守
- 最大监管风险：如果项目主张同时覆盖头皮、面部和长期复发控制，但证据包只支持短期改善，路径会明显失衡

### 判断级别与关键假设

- 当前判断级别：第一轮策略判断或正式策略判断，取决于现有局部耐受和探索性疗效数据
- 关键假设：泡沫剂能在头皮和面部提高可用性，并带来真实临床使用价值
- 高优先级缺口：
  - 目标主张到底是短期改善还是长期控制
  - 头皮与面部位点的一致性证据
  - 长期或复发相关安全与疗效支持

### 保守路径

- 先把适应症主张收窄为较稳妥的短期控制
- 保留更扎实的位点可用性和长期安全支持
- 对双区域扩展更稳

### 激进路径

- 若已有较清晰的探索性疗效和位点适用性信号，可更快进入接近验证性的路径
- 但仍需避免把“起效快”直接等同于“可支持长期价值主张”

### 关键依据类型

- 脂溢性皮炎模块
- FDA 公开审评资料，尤其是 roflumilast foam 相关实践
- 中国局部给药局部起效与改良型或创新型路径的一般规则

## Example Early-Stage Design Detail

### 早期研究总体判断

- 该项目是否需要经典 HV-SAD，要取决于它到底是“已知逻辑的剂型优化”，还是“带来明显新递送和新暴露行为的项目”。
- 如果只是泡沫剂带来更好的头皮可用性和依从性，通常不需要把 SAD 做成主研究。
- 更关键的是尽早回答：
  - 头皮和面部使用是否都可耐受
  - 连续使用下泡沫剂是否真正改善使用便利性
  - 如拟支持慢病复发控制，系统和局部安全是否足够成熟

### 单次给药研究建议

- 若配方或递送方式明显改变暴露特征，可先做简短单次用药耐受和 PK 研究。
- 若系统暴露预期极低且核心问题是部位可用性，单次研究可弱化，不必执着于完整 SAD 叙事。

### 重复给药研究建议

- 该项目更适合以短期重复给药研究作为早期主轴。
- 推荐人群：
  - 若主要想看头皮和面部真实使用场景，优先患者
  - 若面部刺激风险特别担心，也可先用较小健康受试者耐受研究补充
- 关键设计点：
  - 头皮和面部是否同一研究内覆盖，还是分层处理
  - 连续给药后局部刺激、瘙痒、油腻感、可清洗性
  - 如美国优先，还要关注是否需要较真实使用面积下的 PK 或安全确认

### 患者早期信号建议

- 若项目主张“起效快”，建议在早期患者重复给药研究中就收集：
  - 红斑和鳞屑改善方向
  - 瘙痒变化
  - 头皮区域可用性反馈
- 这一步的价值不只是看有没有信号，还在于决定后续项目到底走：
  - 急性控制
  - 复发控制
  - 维持使用

### 转段门槛

- 如果要进入更正式的探索或关键研究，至少应满足：
  - 头皮和面部使用边界已明确
  - 局部耐受对真实用药不构成明显障碍
  - 若声称起效快，已有早期症状或体征改善趋势支持
  - 若声称长期价值，已有继续补长期安全和复发控制证据的清晰计划

## Main Teaching Point

Seborrheic dermatitis strategy becomes much cleaner once the answer explicitly separates acute-control ambition from maintenance or relapse-control ambition.
