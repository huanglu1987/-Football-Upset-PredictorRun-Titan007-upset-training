# Worked Example: Innovative AD Topical Program, Dual Region

## Representative Input

```text
项目代号 ADX-101。新型外用小分子乳膏，用于轻中度特应性皮炎，拟开发中国和 FDA 双区域。当前已有非临床药效、基础安全性和局部耐受数据，但尚无人体 PK。请输出保守路径与激进路径，并重点说明早期临床应如何设计。
```

## Why This Example Matters

This example is useful for:

- true innovative-topical rather than bridge-heavy logic
- dual-region AD strategy with sparse early human data
- proof-of-concept-oriented early development

## What a Strong Answer Should Include

- 明确指出该项目当前更像“创新药第一轮策略判断”，而不是改良型新药桥接题
- 明确区分：
  - 证明人体药理或临床信号
  - 证明局部耐受
  - 证明系统暴露可控
- 说明 AD 项目里瘙痒、慢病长期安全和儿科节奏为什么会影响路径
- 对双区域路径说明：
  - 中国侧更强调整体临床进入依据和临床药理规划前置
  - FDA 侧更强调早期局部和系统暴露控制、以及与近期公开审评实践的一致性

## Example Output Shape

### 项目结论摘要

- 当前更接近：外用创新药路径
- 推荐主路径：偏保守
- 最大监管风险：在没有人体 PK 和早期人体信号时，过早把项目压缩成“接近确证”的开发叙事

### 判断级别与关键假设

- 当前判断级别：第一轮策略判断
- 关键假设：该新分子可在局部达到有效暴露，同时系统暴露保持在可控范围
- 高优先级缺口：
  - 人体 PK / 最大使用或患者 PK 证据
  - 初步暴露-效应或早期临床信号
  - 长期局部耐受与慢病使用可行性判断

### 保守路径

- 先建立更完整的早期临床药理包
- 优先回答强度、频次、局部耐受和系统暴露问题
- 再进入更正式的 POC 或探索性有效性研究

### 激进路径

- 若早期患者研究已能同时获得局部耐受、患者 PK 和可解释的临床信号，可考虑把一部分早期探索整合
- 但激进路径前提是“人体证据开始收敛”，不能只靠非临床推演

### 关键依据类型

- 中国创新药临床药理与暴露-效应逻辑
- 中国局部给药局部起效指导原则
- FDA Exposure-Response / adaptive design / topical review practice
- AD 模块中对慢病、安全和瘙痒终点的要求

## Example Early-Stage Design Detail

### 早期研究总体判断

- 不建议直接从“接近 POC 的患者有效性研究”起跳。
- 更合理的路径是：`先健康受试者，再尽早进入患者`。
- 原因不是机械沿用传统 I 期，而是当前有 3 个问题需要分层解决：
  - 首人体单次和短期重复给药的系统暴露是否可控
  - 完整皮肤与 AD 病损皮肤条件下的暴露和局部耐受是否明显不同
  - 瘙痒和皮损改善的早期信号能否支持后续浓度和频次选择

### SAD 设计建议

- 目标：先回答单次给药后的局部耐受、系统 PK 和剂量递增可行性。
- 推荐人群：健康受试者优先。
- 设计框架：随机、双盲、载体对照或安慰剂对照的序贯队列设计。
- 递增维度：优先考虑 `浓度` 和 `给药面积`，而不是只写“高低剂量”。
- 关键观察：
  - 局部刺激、灼热、刺痛、红斑
  - 单次给药后系统 PK
  - 必要时探索局部样本或生物标志物
- 升阶门槛：
  - 局部耐受可接受
  - 无临床意义的系统安全性风险
  - 暴露水平仍处于可接受窗口

### MAD 或重复给药研究建议

- 不建议只停留在 HV-SAD 后直接进入正式 POC。
- 建议增加一个短期重复给药研究，用来回答：
  - 预设给药频次在连续使用下是否可耐受
  - 是否存在系统蓄积趋势
  - 重复给药后局部刺激是否加重
- 人群选择：这一步更建议转入患者，而不是继续只在健康受试者中完成。
- 原因：AD 病损皮肤可能改变吸收，也更能体现真实使用下的局部耐受。

### 患者 PK / 早期信号整合建议

- 建议在轻中度 AD 患者中开展小样本、重复给药的患者 PK + 早期信号研究。
- 这个研究的角色不是“直接确证有效”，而是：
  - 确认病损皮肤条件下的暴露
  - 观察瘙痒和 IGA 或 EASI 类指标的方向性变化
  - 帮助锁定后续 POC 研究的浓度和频次
- 若存在较大给药面积或系统暴露担忧，可在这一阶段加入 max-use 或较大面积用药子设计。

### 转段门槛

- 只有当以下 4 点基本成立时，才建议转入更正式的 POC 或 II 期探索：
  - 至少 1 个候选浓度和频次具备可辩护依据
  - 患者条件下系统暴露可控
  - 局部耐受未成为主要限制因素
  - 瘙痒或皮损终点出现可解释的早期改善信号

## Main Teaching Point

For innovative topical AD programs, the main question is not what can be borrowed, but how quickly the program can earn credible human proof of concept without losing exposure-control discipline.
