# Worked Example: Innovative AD Topical Program, Dual Region

## Representative Input

```text
项目代号 ADX-101。新型外用小分子乳膏，用于轻中度特应性皮炎，拟开发中国和 FDA 双区域。当前已有非临床药效、基础安全性和局部耐受数据，但尚无人体 PK。请输出保守路径与激进路径，并重点说明早期临床应如何设计。
```

## Why This Example Matters

This example is useful for:

- true innovative-topical rather than bridge-heavy logic
- dual-region AD strategy with sparse early human data
- proof-of-concept-oriented early development

## What a Strong Answer Should Include

- 明确指出该项目当前更像“创新药第一轮策略判断”，而不是改良型新药桥接题
- 明确区分：
  - 证明人体药理或临床信号
  - 证明局部耐受
  - 证明系统暴露可控
- 说明 AD 项目里瘙痒、慢病长期安全和儿科节奏为什么会影响路径
- 对双区域路径说明：
  - 中国侧更强调整体临床进入依据和临床药理规划前置
  - FDA 侧更强调早期局部和系统暴露控制、以及与近期公开审评实践的一致性

## Example Output Shape

### 项目结论摘要

- 当前更接近：外用创新药路径
- 推荐主路径：偏保守
- 最大监管风险：在没有人体 PK 和早期人体信号时，过早把项目压缩成“接近确证”的开发叙事

### 判断级别与关键假设

- 当前判断级别：第一轮策略判断
- 关键假设：该新分子可在局部达到有效暴露，同时系统暴露保持在可控范围
- 高优先级缺口：
  - 人体 PK / 最大使用或患者 PK 证据
  - 初步暴露-效应或早期临床信号
  - 长期局部耐受与慢病使用可行性判断

### 保守路径

- 先建立更完整的早期临床药理包
- 优先回答强度、频次、局部耐受和系统暴露问题
- 再进入更正式的 POC 或探索性有效性研究

### 激进路径

- 若早期患者研究已能同时获得局部耐受、患者 PK 和可解释的临床信号，可考虑把一部分早期探索整合
- 但激进路径前提是“人体证据开始收敛”，不能只靠非临床推演

### 关键依据类型

- 中国创新药临床药理与暴露-效应逻辑
- 中国局部给药局部起效指导原则
- FDA Exposure-Response / adaptive design / topical review practice
- AD 模块中对慢病、安全和瘙痒终点的要求

## Main Teaching Point

For innovative topical AD programs, the main question is not what can be borrowed, but how quickly the program can earn credible human proof of concept without losing exposure-control discipline.
