# Worked Example: AGA, Topical Finasteride Risk Case

## Representative Input

```text
请设计一个 topical finasteride 用于 AGA 的临床开发策略，目标市场为美国。希望尽可能走激进路径，缩短开发时间。
```

## Why This Example Matters

This example is useful for:

- high-risk AGA strategy
- cases where the user explicitly asks for the aggressive route
- cases where FDA safety signaling should override optimistic compression

## What a Strong Answer Should Include

- 直接指出当前不存在可直接借用的 FDA 已批准 topical finasteride 成熟先例
- 明确强调系统暴露、次级接触暴露和生殖安全沟通
- 即使给出激进路径，也应说明它的前提条件非常强，且监管风险明显高于其他优先适应症
- 不应把这个项目写成“已有口服先例，因此桥接就足够”

## Example Output Shape

### 项目结论摘要

- 当前更接近：高不确定性改良型新药 / 505(b)(2) 探索路径
- 推荐主路径：保守路径
- 最大监管风险：系统暴露和 transfer exposure 风险会直接影响 FDA 可接受性

### 判断级别与关键假设

- 当前判断级别：第一轮策略判断
- 关键假设：可将局部暴露与系统暴露控制在足以支持风险可接受的范围
- 高优先级缺口：max-use PK、长期安全、transfer exposure 风险控制

### 保守路径

- 先做更扎实的暴露控制与安全表征
- 再决定是否进入较大规模疗效验证

### 激进路径

- 只有在早期暴露和安全信号都明显可控时才可考虑
- 即便如此，也应明确这是争取性路径，不是常规低风险路径

### 关键依据类型

- FDA 官方安全通告
- 暴露-效应指导原则
- AGA 模块中的高不确定性逻辑

## Main Teaching Point

In topical-finasteride-like AGA programs, the aggressive path should remain visibly risky even when the user asks for speed.
