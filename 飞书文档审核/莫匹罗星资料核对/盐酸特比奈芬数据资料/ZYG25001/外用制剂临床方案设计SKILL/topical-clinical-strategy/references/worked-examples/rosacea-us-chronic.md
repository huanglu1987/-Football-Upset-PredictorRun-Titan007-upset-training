# Worked Example: Rosacea, US-First Chronic-Use Path

## Representative Input

```text
我要做一个玫瑰痤疮外用泡沫剂，美国路径优先。活性成分已有其他剂型上市经验，但新剂型可能改善局部刺激。请输出保守和激进两条开发路径。
```

## Why This Example Matters

This example is useful for:

- papulopustular rosacea
- US-first strategy
- formulation optimization with chronic-use implications

## What a Strong Answer Should Include

- 明确聚焦丘疹脓疱型 rosacea，而不是混入持续性红斑逻辑
- 明确长期安全为何仍然重要，即使系统暴露可能较低
- 对 FDA 审评实践的使用应以 Epsolay、Zilxi 这类公开案例为支撑，而不是泛泛而谈
- 说明新剂型改善局部刺激是一种临床优势假设，但仍需通过设计来验证

## Example Output Shape

### 项目结论摘要

- 当前更接近：改良型新药 / 505(b)(2) 路径
- 推荐主路径：中度偏保守
- 最大监管风险：过早弱化长期安全和耐受性要求

### 判断级别与关键假设

- 当前判断级别：第一轮策略判断或正式策略判断，取决于现有局部耐受和 PK 数据是否充分
- 关键假设：新泡沫剂确实能改善局部刺激，而不牺牲暴露一致性
- 高优先级缺口：长期安全包和剂型转换后耐受性证据

### 保守路径

- 剂量或剂型探索更充分
- 较稳健的 12 周确证研究
- 保留长期安全延伸

### 激进路径

- 若系统暴露很低且局部耐受改善信号明确，可压缩部分早期系统研究
- 但不建议轻易压缩长期安全

### 关键依据类型

- FDA 公开审评案例：Epsolay、Zilxi
- FDA max-use PK / topical review practice

## Main Teaching Point

In chronic rosacea development, low PK concern does not automatically remove long-term safety pressure.
