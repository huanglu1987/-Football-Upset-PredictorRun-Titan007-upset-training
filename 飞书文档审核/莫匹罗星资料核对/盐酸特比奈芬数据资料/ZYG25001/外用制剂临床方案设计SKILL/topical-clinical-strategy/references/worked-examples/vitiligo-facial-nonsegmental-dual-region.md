# Worked Example: Facial-Focused Nonsegmental Vitiligo, Dual Region

## Representative Input

```text
项目代号 VT-201。某外用小分子乳膏拟用于 nonsegmental vitiligo，计划中国和 FDA 双区域开发。当前已有非临床药效、基础安全性和局部刺激数据，但尚无人体 PK。团队希望优先支持面部复色价值，后续再考虑扩大到更广泛身体部位。请输出保守路径与激进路径，并重点说明早期临床设计。
```

## Why This Example Matters

This example is useful for:

- newly added vitiligo indication work
- face-first positioning rather than immediate broad-body claim
- innovative or semi-innovative topical logic with limited early human data

## What a Strong Answer Should Include

- 明确指出项目应先把主张收窄为 `nonsegmental vitiligo + 面部优先`
- 解释为什么白癜风不能按短周期炎症病去写早期开发节奏
- 说明面部复色、全身复色和维持治疗在研究时长、终点和暴露控制上的差别
- 对双区域路径说明：
  - FDA 侧有较强的面部复色先例和公开审评逻辑可参考
  - 中国侧更需要把局部给药、暴露控制和整体临床开发节奏讲清楚

## Example Output Shape

### 项目结论摘要

- 当前更接近：外用创新药或创新性较强的局部起效路径
- 推荐主路径：偏保守，但允许面部优先的中度激进路径
- 最大监管风险：如果早期就把目标扩到较大 body surface area，而没有患者 PK 或 treated-area 暴露控制，路径会失稳

### 判断级别与关键假设

- 当前判断级别：第一轮正式策略判断
- 关键假设：局部暴露足以支持复色，但系统暴露和较大面积使用风险仍可控
- 高优先级缺口：
  - 患者条件下的系统暴露数据
  - 面部目标人群的可耐受性和慢性使用可行性
  - treatment duration 与复色终点的现实窗口

### 保守路径

- 先把目标限定在面部优先的 nonsegmental vitiligo
- 先建立人体 PK 和短期重复给药可耐受性
- 再进入较正式的患者探索研究，并用更长治疗周期判断复色趋势

### 激进路径

- 若早期患者研究已能同时支持面部可耐受性、患者 PK 和明确方向性复色信号，可更快进入接近 POC 的探索路径
- 但不建议在证据不成熟时同步追求“面部 + 全身”双主张

### 关键依据类型

- vitiligo indication module
- FDA Opzelura vitiligo public review logic
- China local-acting topical, innovation-method, and exposure-response guidance

## Example Early-Stage Design Detail

### 早期研究总体判断

- 该项目不建议直接跳到“大面积 body involvement 的长期有效性研究”。
- 更合理的路径是：`简短健康受试者首人体 + 尽早转入患者重复给药研究`。
- 原因有 3 个：
  - 面部长期外用的局部耐受必须尽早看清
  - 若未来可能扩展到更大 treated BSA，就不能完全跳过人体暴露控制
  - 白癜风的有效性窗口较长，早期患者研究更适合回答“是否值得继续”，而不是短期确证

### SAD 设计建议

- 若分子机制存在系统免疫学或类 JAK 风险担忧，建议保留一个简短 HV-SAD。
- 目标：
  - 回答首人体单次给药后的系统 PK
  - 回答面部或有限面积给药时的局部耐受
  - 为后续患者研究中的浓度和频次选择提供上限依据
- 设计框架：随机、双盲、载体对照或安慰剂对照、序贯升阶。
- 递增维度：
  - `浓度`
  - `给药面积`
  - 必要时 `频次`
- 若非临床和体外经皮渗透已强烈支持低系统暴露，HV-SAD 可以简化，但不宜完全缺位。

### MAD 或重复给药研究建议

- 该项目的重复给药主研究更建议放在患者中，而不是只在健康受试者中完成。
- 推荐在 `nonsegmental vitiligo 且面部受累明显` 的患者中开展小样本、重复给药研究。
- 目标：
  - 确认面部连续用药的可耐受性
  - 获取患者条件下 PK
  - 观察早期复色方向性信号，而不是追求最终确证
- 若后续打算扩大到身体其他部位，可在这一阶段设置 `treated-area expansion` 子设计，但不要一开始就做得过宽。

### 患者早期信号建议

- 建议尽早在患者研究中收集：
  - 面部复色评分趋势
  - 标准化摄影
  - 患者对外观改善的主观判断
  - 若适用，total-body 或非面部区域的探索性变化
- 这些信号的主要价值是帮助回答：
  - 是否继续坚持面部优先主张
  - 是否有必要扩大 body-area ambition
  - 后续研究是更偏 induction，还是开始规划 maintenance

### 转段门槛

- 只有当以下条件基本成立时，才建议进入更正式的探索或关键研究：
  - 至少 1 个浓度和频次具备可辩护依据
  - 面部连续给药局部耐受可接受
  - 患者 PK 或 treated-area 暴露控制已基本明朗
  - 已观察到可解释的复色方向性信号

## Main Teaching Point

Vitiligo programs become much cleaner when the answer resists broad body-area ambition too early and deliberately earns a facial-first claim with exposure control.
