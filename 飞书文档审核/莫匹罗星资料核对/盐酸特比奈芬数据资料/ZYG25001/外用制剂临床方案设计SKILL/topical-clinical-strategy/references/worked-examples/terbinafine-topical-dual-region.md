# Worked Example: Terbinafine Topical, Dual Region

## Representative Input

```text
请为盐酸特比奈芬外用乳膏设计中国和 FDA 的临床开发策略。项目希望兼顾开发效率，但也要有一条更稳妥的保守路径。适应症是浅表真菌感染，现有基础主要来自口服和既往外用临床经验。
```

## Why This Example Matters

This example is useful for:

- business-near superficial fungal programs
- same active ingredient with strong prior experience
- dual-region strategy with practical bridge questions

## What a Strong Answer Should Include

- 明确指出该项目与“全新活性成分外用开发”不同，已有成分认知会明显影响激进路径可行性
- 仍然要区分已有外用经验能支持什么、不能支持什么
- 对浅表真菌感染的真菌学终点、临床终点和综合治愈终点分别阐明
- 对中国和 FDA 路径分别说明：
  - 中国侧如何体现临床优势或开发价值
  - FDA 侧如何利用既有经验、同时避免把 PSG 或仿制思路直接当成新药路径

## Example Output Shape

### 项目结论摘要

- 当前更接近：已知活性成分改良或差异化路径
- 推荐主路径：中度偏激进，但以保守路径作为主备份
- 最大监管风险：已有经验虽多，但缩疗程、差异化剂型或新标签主张仍需实证支持

### 判断级别与关键假设

- 当前判断级别：正式策略判断
- 关键假设：既有口服和外用经验足以降低系统性不确定性，但不能替代当前产品的局部开发论证
- 高优先级缺口：目标剂型和标签主张对应的疗效与治愈窗口证据

### 保守路径

- 保留更稳健的真菌学和随访设计
- 不过度压缩疗程
- 更适合双区域同步推进

### 激进路径

- 更充分利用既有 terbinafine 认知，减少系统研究负担
- 若产品主张变化有限，可更早进入接近验证性的路径

### 关键依据类型

- 中国局部起效与改良型新药指导原则
- FDA topical antifungal 背景与 PSG 支持逻辑
- 同活性成分已知经验

## Main Teaching Point

Strong historical familiarity with terbinafine can lighten the path, but it does not erase the need to prove the new product’s exact clinical claim.
