# ClinicalTrials.gov Strategy Layer

## Purpose

Use this file when same-ingredient, same-target, same-indication, or same-formulation trial-design precedent would materially improve the strategy answer.

This layer is meant to strengthen design judgment, not to replace regulatory guidance or FDA public review logic.

## What This Layer Adds

ClinicalTrials.gov is useful for finding:

- ongoing or completed studies with the same active ingredient
- same-target studies without the same ingredient
- same-indication studies with similar endpoint strategy
- similar formulation programs such as foam, spray, or film-forming topical products
- operational design patterns that may not be fully visible in formal guidance

Typical fields worth extracting:

- study phase
- treatment duration
- primary and secondary endpoints
- key inclusion and exclusion patterns
- comparator choice
- follow-up window
- long-term extension design
- pediatric or adolescent cohorts
- posting of results or key outcome summaries

## When To Use It

Reach for this layer when:

- the project is oral-to-topical and direct topical precedent is weak
- the target or mechanism has little approved-topical history
- the answer needs help choosing duration, comparator, or endpoint timing
- the user explicitly asks about same-target or same-class precedent
- the answer would benefit from recent ongoing-study pattern checks

## Evidence Position

Use this layer after:

1. formal CDE/NMPA and FDA guidance
2. FDA public reviews, labels, and review packages

Use this layer alongside:

- PSG, when FDA topical technical expectations matter
- curated case summaries, when a faster precedent scan is helpful

If ClinicalTrials.gov patterns conflict with higher-tier regulatory sources, the regulatory sources win.

## How To Search

Prefer combinations of these search axes:

- active ingredient
- target or mechanism
- indication
- dosage form
- sponsor
- phase
- status

Typical search questions:

- Are there Phase 2 or Phase 3 topical studies in the same indication?
- What endpoint timing windows are being used?
- Are studies using vehicle control, active control, or both?
- Is long-term safety or extension follow-up common?
- Are adolescents included from the start or only later?

## How To Interpret Results

Treat registry data as a design-pattern source, not a proof source.

Important limits:

- a registered design is not the same as an FDA-accepted design
- a planned endpoint is not proof that the endpoint succeeded
- some records are outdated or incompletely updated
- missing results do not mean the study failed, and posted results do not replace review documents

Higher-confidence registry signals usually come from:

- completed Phase 2 or Phase 3 studies
- records with posted results
- multiple studies converging on similar design choices
- recent updates rather than stale registrations

## Recommended Output Use

When this layer influences the answer, say so explicitly:

- evidence type: public clinical trial registry precedent
- evidence point: what design pattern was observed
- applicability condition: why that pattern is relevant to the current project

Do not write as if ClinicalTrials.gov creates a regulatory requirement.
