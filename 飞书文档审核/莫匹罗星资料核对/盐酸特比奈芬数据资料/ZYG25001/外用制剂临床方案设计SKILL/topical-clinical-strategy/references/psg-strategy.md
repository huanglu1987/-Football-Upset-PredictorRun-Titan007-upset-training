# PSG Strategy

## Purpose

Use this file to decide when PSG should materially influence the answer and when it should stay in a supporting role.

## Official Sources

- PSG portal:
  `https://www.fda.gov/drugs/guidances-drugs/product-specific-guidances-generic-drug-development`
- FDA explainer on PSG:
  `https://www.fda.gov/drugs/development-approval-process-drugs/abcs-product-specific-guidances`
- OTC topical antifungal order:
  `https://www.accessdata.fda.gov/drugsatfda_docs/omuf/OTC%20Monograph_M005-Topical%20Antifungal%20drug%20products%20for%20OTC%20Human%20Use%2012.16.2021.pdf`

## Role of PSG

Treat PSG as:

- a technical support layer
- a strong clue about FDA’s expectations for complex topical development mechanics
- a weak source for direct innovative-drug or 505(b)(2) pathway mandates

Do not treat PSG as:

- a substitute for FDA pathway guidance
- a substitute for public review practice
- a binding rule for innovative or improved-new-drug strategy

## When PSG Matters Most

PSG matters most when the strategy question turns on:

- dosage-form sameness or near-sameness
- formulation complexity
- Q1/Q2/Q3 concerns
- IVRT or IVPT expectations
- whether the product resembles a generic-development question more than a novel clinical-development question

## When PSG Matters Less

PSG matters less when the main question is:

- whether the product is innovative or improved
- whether a bridge to prior clinical knowledge is adequate
- whether chronic-use safety can be reduced
- whether a new clinical claim is acceptable

In those situations, FDA guidance and public review documents should usually carry more weight.

## How to Search PSG

When live browsing is needed:

1. search the active ingredient
2. search the dosage form
3. check whether the RLD or a closely related topical product exists
4. note whether the PSG is current or revised
5. extract only the parts relevant to:
   - local delivery or sameness logic
   - study type
   - technical evidence expectations

## Indication Mapping

### Acne

Use PSG mainly to refine:

- vehicle and formulation comparability logic
- technical support for semisolid products
- whether local endpoint logic resembles existing topical comparators

Weighting rule:

- FDA acne guidance and public review practice usually outrank PSG for overall clinical strategy.

### Rosacea

Use PSG selectively.

Main uses:

- supportive formulation logic
- technical expectations if the product closely resembles an existing topical comparator

Weighting rule:

- public review packages such as Zilxi and Epsolay usually matter more than PSG for overall strategy.

### Superficial Fungal Infection

This is the highest-value PSG module among the four priority indications.

Use PSG to refine:

- endpoint mechanics
- timing logic
- formulation comparability
- technical evidence package

Also check the OTC topical antifungal order when the active or claim overlaps OTC antifungal space.

### AGA

PSG usually has limited direct value unless the active and dosage form closely match a marketed topical product.

Weighting rule:

- for minoxidil-like logic, use PSG only as supportive background
- for topical finasteride-like logic, public FDA safety signals and exposure concerns matter much more than PSG

## Output Rule

If PSG influences the answer, say so explicitly using language like:

- `该建议部分参考了 PSG 中反映的外用复杂制剂技术预期，但 PSG 本身不直接决定本项目的创新药/505(b)(2) 路径。`

This prevents overstating PSG authority.
