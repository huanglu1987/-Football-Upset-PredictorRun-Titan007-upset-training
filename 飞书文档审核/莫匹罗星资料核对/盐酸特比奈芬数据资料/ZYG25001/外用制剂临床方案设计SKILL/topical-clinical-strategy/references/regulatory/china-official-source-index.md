# China Official Source Index

## Purpose

This file is the public-distribution-safe source index for China regulatory materials used by this skill.

The repo does not rely on author-local raw PDF paths for normal use.
Instead, the skill should use:

- repo-bundled summary cards for day-to-day strategy work
- this index when official-source provenance or current verification is needed

## Retrieval Rule

For each title below:

1. Start with the CDE portal: `https://www.cde.org.cn/`
2. Search the exact Chinese title on the official site
3. If needed, cross-check on the NMPA portal: `https://www.nmpa.gov.cn/`

When answering a user question, the repo summary cards remain the default working layer. Use live official retrieval when the user asks for current, latest, or citation-ready support.
