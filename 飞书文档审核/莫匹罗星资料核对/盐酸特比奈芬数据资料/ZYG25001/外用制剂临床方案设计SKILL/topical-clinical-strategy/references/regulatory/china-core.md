# China Core Regulatory Cards

## Purpose

Use these cards to anchor China-side strategy before you infer anything from FDA precedent.

## 1. 局部给药局部起效的药物临床试验技术指导原则

Official source:

- CDE/NMPA official publication. See `references/regulatory/china-official-source-index.md` and retrieve by exact title.

Use for:

- deciding whether the product fits the locally acting topical framework
- separating innovative, improved, and generic logic
- judging whether local PK, local PD, bridge work, or equivalence work should appear

Key takeaways:

- 局部给药局部起效产品需同时关注局部和全身耐受性、安全性、局部和系统 PK、局部 PD、剂量探索。
- 创新药、改良型新药、仿制药的开发逻辑不相同，不能直接混用。
- 对已有系统给药或已有局部给药上市基础的同活性成分，桥接是核心议题。

## 2. 化学药品改良型新药临床试验指导原则

Official source:

- CDE/NMPA official publication. See `references/regulatory/china-official-source-index.md` and retrieve by exact title.

Use for:

- defining the clinical-advantage question
- deciding whether the improved product must prove better efficacy, better safety, or better adherence

Key takeaways:

- 中国改良型新药强调“明确临床优势”，不能只停留在剂型变化本身。
- 若主张改善安全性或提高依从性，需要让临床设计真正对应这一主张。
- 对口服改外用、已知活性成分新剂型等项目，临床试验设计要围绕改良目标组织。

## 3. 化学药改良型新药临床药理学研究技术指导原则（试行）

Official source:

- CDE/NMPA official publication. See `references/regulatory/china-official-source-index.md` and retrieve by exact title.

Use for:

- deciding whether bridge PK is enough
- deciding whether fuller clinical pharmacology work is needed
- determining how much systemic exposure matters

Key takeaways:

- 对存在系统暴露的改良型新药，临床药理学研究有时是关键研究事项。
- 当改良改变了 PK 特征、处方工艺或给药途径时，不能默认沿用原产品临床药理认知。
- 暴露、PD、剂量-暴露-效应关系都可直接影响后续临床开发路径。

## 4. 药物暴露-效应关系研究技术指导原则

Official source:

- CDE/NMPA official publication. See `references/regulatory/china-official-source-index.md` and retrieve by exact title.

Use for:

- justifying target strength, frequency, and regimen
- deciding whether dose-ranging can be compressed or not
- strengthening high-uncertainty modules such as AGA or oral-to-topical switches

Key takeaways:

- 暴露-效应关系是支持有效性和安全性评价、优化给药方案的核心证据之一。
- 早期研究中的暴露-效应分析，即使在经典统计学不显著时，也可能帮助确认药理信号。
- 当浓度或频次依据较弱时，应优先考虑暴露-效应逻辑，而不是直接放弃路径设计。

## 5. 药物临床试验申请临床评价技术指导原则

Official source:

- CDE/NMPA official publication. See `references/regulatory/china-official-source-index.md` and retrieve by exact title.

Use for:

- checking whether current evidence is enough to support next-step clinical entry
- identifying risk-control expectations for CTA stage

Key takeaways:

- 临床需求、前期研究数据、整体研发计划和具体方案要一起评价。
- 关键方案要素应由前期非临床或临床数据支持。
- 对安全性风险控制的准备程度，会直接影响是否适合进入下一阶段。

## Practical Rule

If China is in scope:

1. Start with this file.
2. Build strategy around local-acting and improved-new-drug logic first.
3. Use FDA precedent only after the China-side anchor is clear.
4. If public-source verification is needed, use `references/regulatory/china-official-source-index.md` rather than any author-local file path.
