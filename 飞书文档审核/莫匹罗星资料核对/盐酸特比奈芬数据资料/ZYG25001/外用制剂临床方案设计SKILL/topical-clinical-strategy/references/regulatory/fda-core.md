# FDA Core Regulatory Cards

## Purpose

Use these cards to anchor FDA-side strategy before you rely on product-specific public review examples.

## 1. Determining Whether to Submit an ANDA or a 505(b)(2) Application

Official link:

- `https://www.fda.gov/regulatory-information/search-fda-guidance-documents/determining-whether-submit-anda-or-505b2-application`

Use for:

- deciding whether a topical product really fits 505(b)(2) logic instead of ANDA logic
- explaining why formulation, strength, route, active combination, or data needs push a product beyond ANDA territory

Key takeaways:

- Products that differ materially from the RLD or require data beyond what fits an ANDA often point toward 505(b)(2).
- This guidance is foundational when the user is weighing improved topical products against generic pathways.

## 2. Maximal Usage Trials for Topically Applied Active Ingredients Being Considered for Inclusion in an Over-The-Counter Monograph

Official link:

- `https://www.fda.gov/regulatory-information/search-fda-guidance-documents/maximal-usage-trials-topically-applied-active-ingredients-being-considered-inclusion-over-counter`

Use for:

- thinking about max-use PK or MUsT logic
- framing systemic-exposure risk in locally acting topical products

Key takeaways:

- MUsT is a standard approach to assess in vivo bioavailability for topical products intended for local therapeutic effect.
- Even though this guidance is framed for OTC monograph development, the study logic is highly relevant when systemic exposure concerns may influence safety expectations.

## 3. Product-Specific Guidances for Generic Drug Development

Official link:

- `https://www.fda.gov/drugs/guidances-drugs/product-specific-guidances-generic-drug-development`

Use for:

- understanding FDA’s technical expectations for complex topical products
- harvesting supportive logic around IVRT, IVPT, Q1/Q2/Q3, and endpoint structure

Key takeaways:

- PSG is a supportive technical layer, not the controlling rule set for innovative or 505(b)(2) strategy.
- It is especially useful when the development question turns on formulation sameness, local equivalence logic, or endpoint operationalization.

## 4. Acne Vulgaris: Establishing Effectiveness of Drugs Intended for Treatment

Official link:

- `https://www.fda.gov/regulatory-information/search-fda-guidance-documents/acne-vulgaris-establishing-effectiveness-drugs-intended-treatment`

Use for:

- acne efficacy endpoint selection
- confirming lesion-count and global-assessment logic

Key takeaways:

- FDA has an explicit acne efficacy framework, so acne strategy should be more standardized than many other dermatology indications.
- Use this guidance before borrowing endpoint packages from individual review examples.

## 5. AGA Safety Signal for Topical Finasteride

Official link:

- `https://www.fda.gov/drugs/human-drug-compounding/fda-alerts-health-care-providers-compounders-and-consumers-potential-risks-associated-compounded`

Use for:

- AGA projects involving topical finasteride-like logic
- explaining why local application does not erase systemic and secondary-exposure concerns

Key takeaways:

- As of April 22, 2025, FDA stated there is no FDA-approved topical formulation of finasteride.
- FDA described both systemic adverse-event concern and transfer-exposure concern for compounded topical finasteride products.

## Practical Rule

If FDA is in scope:

1. Start with pathway guidance and MUsT logic.
2. Use PSG as a supportive technical layer.
3. Use public review cases to understand what FDA accepted in practice.
