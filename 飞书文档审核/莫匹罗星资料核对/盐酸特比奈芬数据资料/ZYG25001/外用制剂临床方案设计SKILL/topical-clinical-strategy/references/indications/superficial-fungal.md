# Superficial Fungal Module

## Scope

Use this module for superficial fungal skin infections. In first-pass strategy work, distinguish at least:

- tinea pedis
- tinea corporis / cruris

Do not assume one timeline or endpoint package fits all superficial fungal programs.

## Priority Sources

Read:

- relevant FDA PSG for the specific active and indication when applicable
- FDA antifungal administrative or guidance materials when relevant
- China local-acting topical guidance

In this module, PSG has higher practical value than in many other modules, but it is still a supportive source rather than a controlling rule for innovative or 505(b)(2) programs.

## Core Endpoint Logic

Separate three endpoint layers:

- mycological endpoint
- clinical endpoint
- complete or composite cure endpoint

Also define:

- treatment-end assessment
- follow-up cure assessment

## Early-Stage Focus

Pay special attention to:

- organism confirmation
- lesion site and extent
- recurrence timing
- local tolerability
- whether systemic exposure is essentially irrelevant or still needs a bridge

## Confirmatory Pattern

Common strategy elements:

- strong microbiology definition
- clear cure-window timing
- careful handling of residual signs vs persistent infection
- vehicle control or standard-treatment context depending on the program type

## Conservative Path Triggers

Lean conservative when:

- the target indication is broad
- recurrence risk is material
- the cure definition is still unstable
- the program is trying to shorten treatment duration without solid support

## Aggressive Path Triggers

Lean aggressive when:

- same active or class precedent is strong
- systemic exposure is negligible
- the main claim is operational improvement rather than a broad efficacy departure

## Common Failure Points

- collapsing mycological cure and clinical cure into one vague endpoint
- ignoring follow-up timing
- over-using PSG language without adjusting for innovative or improved-drug context
