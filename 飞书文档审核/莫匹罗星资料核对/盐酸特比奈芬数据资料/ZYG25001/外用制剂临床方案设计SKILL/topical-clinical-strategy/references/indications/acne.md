# Acne Module

## Scope

Use this module for acne vulgaris, especially when the product targets inflammatory and non-inflammatory facial lesions.

## Priority Sources

Read:

- FDA acne guidance on establishing effectiveness
- relevant public review documents for acne topical products
- `references/review-cases/topical-fda-cases.md`

## Core Endpoint Logic

Common efficacy anchors:

- inflammatory lesion count change
- non-inflammatory lesion count change
- IGA or EGSS success rate

If the product is a fixed-dose combination or modified combination, actively check whether a component-contribution question exists.

## Early-Stage Focus

Pay special attention to:

- local tolerability
- irritation, dryness, peeling, burning
- need for dose-ranging
- max-use PK or systemic-exposure bridge if the active or class raises exposure concerns

## Confirmatory Pattern

Typical confirmatory strategies often include:

- vehicle-controlled studies
- endpoint sets combining lesion counts and global assessment
- dedicated attention to adolescent populations if the intended label includes them

## Conservative Path Triggers

Lean conservative when:

- the product is a novel combination
- concentration rationale is weak
- irritation burden may be meaningful
- adolescent inclusion is ambitious
- the systemic safety bridge is not yet secure

## Aggressive Path Triggers

Lean aggressive when:

- same ingredient or class precedent is strong
- systemic exposure is expected to be very low
- the main value is improved convenience or tolerability rather than a new efficacy claim

## Common Failure Points

- treating combination logic as if contribution evidence is unnecessary
- underestimating local tolerability as a regulatory issue
- assuming oral or topical class precedent fully replaces product-specific bridge logic
