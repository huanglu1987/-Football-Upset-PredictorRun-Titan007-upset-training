# Case Family: AGA

## Use This Family For

- androgenetic alopecia
- hair-growth optimization programs
- hormone-pathway topical programs with systemic-exposure concern

## Main Sub-Families

### 1. 已知外用基础的优化型

Typical examples:

- `worked-examples/minoxidil-optimization-dual-region.md`

Core question:

- can the new dosage form improve tolerability, convenience, or adherence without losing efficacy basis

### 2. 激素通路高风险型

Typical examples:

- `worked-examples/aga-topical-finasteride-risk.md`

Core question:

- can the program contain systemic exposure and secondary-contact risk strongly enough to justify chronic topical use

### 3. 口服改外用高暴露关注型

Closest current pattern:

- start from `worked-examples/aga-topical-finasteride-risk.md`
- then apply the longer half-life and potentially stronger DHT-suppression concern of dutasteride-like logic

Core question:

- is the topical route truly reducing systemic burden enough to create a defendable clinical advantage

## What Usually Matters Most

- long treatment horizon
- objective hair assessment methods
- local versus systemic exposure balance
- exposure-response support for concentration and frequency
- long-term safety framing

## Typical Early-Study Split

- optimization family:
  - weaken classical SAD
  - strengthen short repeat-dose patient usability and tolerability work
- hormone-pathway risk family:
  - stronger early PK
  - more explicit endocrine and reproductive-risk framing
  - patient repeat-dose PK before broad efficacy ambition

## Fast Routing Rule

If the current product is:

- `minoxidil-like`: start from the optimization family
- `finasteride/dutasteride-like`: start from the hormone-pathway family

## Common Failure Points

- using acne-like short timelines
- under-specifying objective hair measurement
- assuming topical route automatically solves systemic concern
