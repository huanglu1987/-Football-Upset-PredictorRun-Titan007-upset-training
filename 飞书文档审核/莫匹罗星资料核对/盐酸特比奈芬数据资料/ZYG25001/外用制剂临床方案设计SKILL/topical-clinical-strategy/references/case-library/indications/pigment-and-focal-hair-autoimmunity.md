# Case Family: Pigment and Focal Hair Autoimmunity

## Use This Family For

- vitiligo
- alopecia areata
- other autoimmune skin or hair disorders where local proof-of-concept is needed but disease fluctuation complicates signal reading

## Main Sub-Families

### 1. Repigmentation Programs

Closest current module:

- `references/indications/vitiligo.md`

Core question:

- can the product deliver clinically meaningful repigmentation within a realistic treatment horizon and treated-area boundary

### 2. Localized Autoimmune Hair-Regrowth Programs

Closest current module:

- `references/indications/alopecia-areata.md`

Core question:

- can the product generate interpretable regrowth signal in a heterogeneous disease with spontaneous fluctuation

## What Usually Matters Most

- disease subtyping before study design
- site-specific endpoint relevance
- longer observation than classic flare-control dermatoses
- strong imaging or standardized assessment discipline
- avoiding over-broad claims too early

## Fast Routing Rule

If the product is built around repigmentation, start from the vitiligo module.

If the product is built around localized patch regrowth, start from the alopecia areata module.

## Common Failure Points

- overstating early cosmetic or regrowth signal
- mixing subtypes with very different natural history
- using too broad a treatment ambition before the early signal is mature
